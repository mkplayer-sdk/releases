## 0.9.2 (2021.06.11)
Links: [Library Download](https://mkplayer.z13.web.core.windows.net/roku/0.9.2/0.9.2.zip) | [Reference Doc](https://mkplayer.z13.web.core.windows.net/roku/0.9.2/roku.html) | [Roku Reference App](https://mkplayer.z13.web.core.windows.net/roku/0.9.2/0.9.2.zip)

- Integration with NBA Prod C environment and adding the ability to play Live & VOD content
- Added in a library method to update the STS token.
- Fixing an issue related to not setting the library config on the global scope
- Added source options with the ability to so specify the start time offset.
- Added in device profile encryption
- Added in accountId & deviceId for multiple Api calls
- Changed the start and end parameters values in the license url to Live & END in case of Live assets
- Aligned API method and property names with other platforms
- Fixed complete Beacon dispatch on exiting a video
